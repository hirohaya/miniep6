-Foi utilizado o padrão Decorator.

-Com a utilização do decorator, foi possível uma economia de 50% das linhas utilizadas no programa sem a aplicação do padrão, além de uma mais clara compreensão do programa dado a simplificação da estrutura.
 O Decorador foi escolhido como padrão a ser utilizado dada a natureza do programa utilizado no caso de um jogador que pode adquirir uma arma e equipamentos.
